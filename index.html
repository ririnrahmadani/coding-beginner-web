CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan halaman web. CSS mengatur aspek visual seperti warna, ukuran teks, layout, dan gaya elemen-elemen pada halaman web.

HTML merupakan singkatan dari Hypertext Markup Language, yaitu bahasa pemrograman yang digunakan untuk membangun dan memformat halaman web. HTML berfungsi sebagai dasar dalam pembuatan struktur konten dan tata letak sebuah website.

JavaScript adalah bahasa pemrograman tingkat tinggi yang sangat penting dalam pengembangan aplikasi web, juga dapat menambahkan fungsi dan interaktivitas ke halaman web Anda. JavaScript mampu memanipulasi elemen HTML, merespons tindakan pengguna, dan mengontrol perilaku halaman web secara dinamis.

Python adalah bahasa pemrograman tingkat tinggi yang populer dan serbaguna. Dikenal dengan sintaksis yang mudah dibaca dan dipahami, Python menjadi pilihan favorit bagi pemula dalam mempelajari pemrograman.

PHP adalah bahasa pemrograman sisi server yang digunakan secara luas untuk pengembangan aplikasi web. PHP memiliki sintaksis yang mirip dengan bahasa pemrograman C, yang membuatnya relatif mudah dipelajari dan dipahami, terutama jika Anda sudah terbiasa dengan bahasa seperti C, C++, atau Java.
